# Movie_tracker

Сервис для отслеживания просмотренных фильмов и сериалов с использованием TG-бота и аналитикой на Веб-сервисе

## Функции
- Telegram bot для комфортного использования сервиса
- Добалением фильмов и сериалов в библиотеку просмотров
- Добавление в список желаемого
- Веб-аналитика с графиками и статистикой по годам (*для особо любопытных*)
